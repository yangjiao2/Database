To compile the program

1) Navigate to the directory containing the files in command line
2) Type javac -cp mysql-connector-java-5.1.34-bin.jar Main.java
3) Verify compilation, there should now be a Main.class file in the folder


To run the program

1) Type java -cp .;mysql-connector-java-5.1.34-bin.jar Main
2) The program will ask for username/password information to connect to a database
3) After connecting successfully, type help; to see additional commands.
